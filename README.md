# Simple Calculator

This project is a basic, command-line calculator implemented in Python. It prompts the user to enter two numbers and choose an arithmetic operator. It then performs the selected calculation and displays the result.

## üöÄ Features

* **Addition (+)**: Adds two numbers.
* **Subtraction (-)**: Subtracts the second number from the first.
* **Multiplication (*)**: Multiplies the two numbers.
* **Division (/)**: Divides the first number by the second.
* **Modulo (%)**: Performs the remainder (modulus) operation.
* **Error Handling**: Provides a "invalid choice" message if the operator is not one of the five supported options.
